# How to Import Test Products & Collections

## Step 1: Import Products
1. Go to **Shopify Admin > Products**
2. Click **Import** (top right)
3. Upload `products.csv` from this folder
4. Click **Upload and continue**
5. Review the preview, then click **Import products**

## Step 2: Create Collections
After products are imported, create these collections manually:

### Automated Collections (recommended)
Go to **Shopify Admin > Products > Collections > Create collection**

1. **All Products** — Type: Automated, Condition: Product price > 0
2. **Graphic Tees** — Type: Automated, Condition: Product tag = `graphic-tees`
3. **Hoodies** — Type: Automated, Condition: Product tag = `hoodies`
4. **Bottoms** — Type: Automated, Condition: Product tag = `bottoms`
5. **Accessories** — Type: Automated, Condition: Product tag = `accessories`
6. **New Arrivals** — Type: Automated, Condition: Product tag = `new-arrivals`
7. **Sale** — Type: Automated, Condition: Product tag = `sale`
8. **Artist: Kira Voss** — Type: Automated, Condition: Product tag = `kira-voss`
9. **Artist: Marco Diaz** — Type: Automated, Condition: Product tag = `marco-diaz`
10. **Artist: Lena Park** — Type: Automated, Condition: Product tag = `lena-park`
